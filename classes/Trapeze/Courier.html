Class Trapeze::Courier
In: lib/trapeze/courier.rb
Parent: Object

Captures any method call made on it and returns another Trapeze::Courier object. Information about method calls is recorded in hashes. A Trapeze::Courier object can be used to describe the interface that the calling code expects to use.
