Class Trapeze::Command

Encapsulates command-line arguments passed to the Trapeze application for controlling its execution.

Methods

Attributes

args  [R]  The command-line arguments.
valid_options  [R]  The command-line options that are valid, along with default values for those options.

This attribute is a hash in the following form: 

  {'--a-long-option-name'       => ['-a', default_value_of_a],
   '--another-long-option-name' => ['-b', default_value_of_b]}

The long form of each option name is the key to an array containing the short form of the option name and its default value.

Default values are optional — they do not need to be specified, for example: 

  {'--an-option-without-default' => '-o'}

The long form of the option name is the key to the short form of the option name.
