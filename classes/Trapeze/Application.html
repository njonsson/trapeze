new(*args)

Instantiates a new Trapeze::Application, establishing execution options supplied in args. Valid options are as follows: 

  --input-files-pattern PATTERN
  --output-dir          PATH
  --help

The concise forms of these options are as follows: 

  -i PATTERN
  -o PATH
  -h
