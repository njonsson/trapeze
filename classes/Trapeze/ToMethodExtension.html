Module Trapeze::ToMethodExtension
In: lib/trapeze/to_method_extension.rb

Adds methods to String objects for obtaining accordingly named Method objects.

The names of these methods are prefixed with an underscore character in order to reduce the likelihood of method name collisions, given that the purpose of Trapeze is to load and analyze other Ruby programs.

_to_instance_method(type)

Returns the Method object for a String that represents an instance method on type. Raises ArgumentError if type is neither a Class nor a Module.
