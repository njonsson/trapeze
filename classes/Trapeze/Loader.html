Class Trapeze::Loader
In: lib/trapeze/loader.rb
Parent: Object

Loads source code files and retrieves class, module and top-level method definitions contained within them.

Note that source code is loaded in the current Ruby process. Changes to the behavior of core Ruby types may take effect as a result.

Public Instance methods

classes()

Returns classes defined in filenames, calling load! if it has not been called already.

The return value is an array of Class objects with _defined_methods and _defined_instance_methods attributes containing the names of methods defined in the source code files.

The class methods and instance methods indicated do not include inherited methods.
